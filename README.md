```shell
go build
./main -domain-file ./<YourDomainFile>

=======
./main -h
Usage of ./main:
  -domain-file string
    	Path to file with domains (separated by newlines)
  -flagname int
    	help message for flagname (default 1234)
  -interval duration
    	Interval to check domains at (default 12h0m0s)
  -n int
    	help message for flag n (default 1234)
```
